Este repositório representa o meu primeiro projeto de site completo. Ele consolida os conhecimentos iniciais de estruturação e estilização, transformando conceitos básicos em uma interface real e navegável.

---

### O que foi explorado:
* **Layout Design:** Organização visual dos elementos na página.
* **Estilização Avançada:** Uso de cores, fontes personalizadas e espaçamentos (Padding/Margin).
* **Navegação:** Criação de menus e links internos/externos.
* **Responsividade Inicial:** Primeiros passos para garantir que o site se adapte a diferentes telas.
